# 🛠️ Sistema de Chamados de Suporte Técnico

Este é um sistema simples de abertura e gerenciamento de chamados de suporte técnico, desenvolvido com Python (Flask) como primeiro projeto prático de TI.

---

##  Funcionalidades

- Abertura de chamados com nome, descrição e categoria
- Visualização de todos os chamados
- Filtro por status: **Todos**, **Abertos**, **Resolvidos**
- Marcar chamados como resolvidos com um clique
- Autenticação de técnico (login e logout)

---

##  Tecnologias utilizadas

- Python 3 + Flask
- HTML, CSS
- SQLite (banco de dados local)
- Jinja (template engine do Flask)
